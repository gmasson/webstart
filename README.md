# Webstart
Project with utilities to start a new website

## Includes
- Bootstrap (CSS and JS)
- Animate CSS
- Kajado CSS
- Bootstrap Icons
- HTML5 Shiv
- Popper.JS
- VueJS
- JSQuantio
- Support for Progressive Web Apps
- Support for Social Widgets
- Support for Sitemap
- Indenting 4 Tabs
